# Ecoli_YeaWX_population_genomics

Representative scripts for population genomics analysis of the YeaWX trimethylamine-associated module in *Escherichia coli*.

## Minimal example workflow

1. Detect YeaX/YeaW homologs using TBLASTN  
2. Generate genome-level presence/absence matrices  
3. Perform phylogroup and sequence type stratification analyses  
4. Compare ARG burden and MDR prevalence  
5. Analyze plasmid family distributions  

## Requirements

- BLAST+
- Python 
- R
- ResFinder
- PlasmidFinder

## Associated study

Population genomics analysis of the YeaWX trimethylamine-associated module across 9,700 *Escherichia coli* genomes.
